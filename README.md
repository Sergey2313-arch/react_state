# React State

Учебный проект по теме **State**.

Компоненты:

- `Hello` — class component со своим состоянием;
- `Form` — форма с состоянием полей;
- `Range` — слайдер со state;
- `Post` и `Posts` — вывод массива через props.

## Запуск

```bash
npm install
npm start
```
